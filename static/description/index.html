Brings Google Maps features into Odoo Web(Available in China>)

Show all your partners location on google maps

Base on web_google_maps Not just partner(res.partner) model but any model contains geolocation fields

This module brings three features:

  • Allows user to view all partners addresses on google maps.
  • Enabled Google places autocomplete address form into partner form view,
    provide autocomplete feature when you typed an address of partner
  • Routes information


Routes

Distance information between two partners locations

With some controls to display different information of routes

  • Map layers (traffic, transit, bicycle)
  • Travel mode (driving, walking, bicycling, transit)
  • Redirection to google maps website to see the full features

New Widget for Google Place Autocomplete Address Form

Provide autocomplete feature when typing an address

How to use:

  • Active develope mode
  • Goto: Settings->Technical->System Parameters
  • Set "app_google_maps_api_key" to your google map api key
